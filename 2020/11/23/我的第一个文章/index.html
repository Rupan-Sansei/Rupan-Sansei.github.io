用hexo框架创建博客基本框架

注意：如果想用hexo框架，要先下载nodejs和git，因为我电脑之前就安装了，所以就不再多说，如果有不会的可以自行百度一下。那么我们就进入正题

1.安装hexo框架，在本地运行并自己的第一个篇文章

​ 创建一个blog文件夹，并在终端中打开。在终端输入npm install -g hexo-cli全局安装，之后再输入hexo -v看是否成功。

​ 之后输入命令hexo init初始化，文件夹会生成几个文件，如下图。之后输入命令hexo s（hexo server的简写）启动本地服务器，默认情况下，访问网址为： http://localhost:4000/,并在本地浏览器打开。

​ 先CTRL+c关闭服务器，之后输入命令hexo new '我的一个文章'在source/_posts下生成.md文件，打开我的一个文章.md输入内容，最后输入hexo g（hexo generate的简写）生成静态文件（第一次生成静态文件会比较多），再hexo s重新打开本地服务器。

2.在github pages上部署hexo博客

​ 第一步，先在自己的github上创建一个仓库(注意!一定是你的账号名字+.github.io)。第二步，配置_config.yml文件，拉到最后，修改为如下格式

​ 第三步输入npm install hexo-deployer-git --save安装hexo的git插件，最后输入hexo d（hexo deploy的简写）部署网站。

注意了！ 因为我之前就用过git，所以git全局配置账号，e-mail和在github上添加ssh钥匙就不演示了，百度一下就能查的到@-@

​ 部署上线就完成啦~，之后可以在https://yourname.github.io 这个网站看到你的博客了！！（如果https不行就换成http）

现在我们已经完成最基本的博客框架，后面我会一步步优化的博客，随便写下博客记录下，加油!
